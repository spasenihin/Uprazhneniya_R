В репозитории расположен скрипт Upr1.rmd, который парсит сайт https://www.labirint.ru/genres/965/ (иностранные книги), по пяти показателям и сохраняет полученные данные в файл labirint_ru.csv.

Структура файла:
Name - название книги
Izdatel - издатель
Izdatel_series - серия издания
Price - цена книги
Count_photo - количество фото, прилогаемое к книге
